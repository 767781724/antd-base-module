#antd-base-module v1.0.3  

基于antd开发的二次封装组件  
依赖antd、react、react-dom、lodash、react-resizable

##安装  

```  
npm install --save antd-base-module 

yarn add antd-base-module  

```

###Base页面组件

```
 BasePage

 BaseListPage   //base基础页面

 BaseTable      //table表格

 BaseFilter     //搜索状态

 BaseToolbar    //功能按钮

 BaseTabs       //tabs切换

 BaseTablePage  //纯表格页面

```

###数据组件

```
CodeInput       //获取验证码

NumberInput     //数字输入
```
